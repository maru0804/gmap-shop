# gmap-shop

Google mapから特定の情報を取得するAPI

# 開発環境
- Python 3.8.5

0. ファイルをローカルに用意
```zsh
git clone https://github.com/maru0804/gmap-shop.git
cd gmap-shop
```
1. venvで仮想環境を作成し有効化
```zsh
python3 -m venv [newenvname]
source [newenvname]/bin/activate
```
2. 必要なパッケージをインストール
```zsh
pip install -r requirements.txt
```
3. 仮想環境の修了
```zsh
deactivate
```
